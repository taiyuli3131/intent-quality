# Local Loop Fixture

This directory is a synthetic fixture for the local-first public sync, suggestion, profile, and contribution review loop.

The nested `.intent-quality/` tree is intentionally committed here as example data. It is not real project runtime state, does not contain private workspace data, and should not be treated as an accepted local profile, contribution submission, public-sample adoption, or applied suggestion.

Real project-local runtime state belongs in the repository root `.intent-quality/` directory, which is ignored by Git.

The fixture demonstrates that public samples and suggestions remain:

- local;
- pending or untrusted by default;
- reviewable before adoption;
- blocked from automatic profile, rule, dataset, casebook, rubric, contribution, or public-sample mutation.
